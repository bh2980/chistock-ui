chistock-ui는 chistock을 위한 디자인 시스템 라이브러리입니다.  
<br/>

- [ ] 디자인 시스템과 스토리북을 통한 CDD 프론트엔드 개발
- [ ] D3.js 혹은 SVG를 이용한 데이터 시각화
- [ ] 테스트 라이브러리들을 이용한 테스트 코드 작성
- [ ] NPM 라이브러리 배포

<br/>
를 목표로 삼아 개발했습니다.

<br/><br/>
